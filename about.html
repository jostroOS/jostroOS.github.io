Project Info

Jostro OS 2.0 is a heavyweight surfaceflinger based alternative to Mac OS On Raspberry Pi with multiple dissadvantages, such as:

  • Being built off Red Star OS for maximum tracking
  • Using WindowManager for a low-performance surfaceflinger desktop, hogging extra system resources
  • Pi-Apps and PiKISS pre-removed so you can't get everything started instantly without delay
  • Space Cadet Pinball pre-removed for you, cry
