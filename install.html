Install Instructions

Follow these easy steps to install Jostro OS on your device:

1. Download the latest image from our Home Page.

2. Flash the image to your device using Raspberry Pi Imager or Balena Etcher.

3. Boot it up and it's done! Enjoy!

(Note that on the first boot you might see the screen flash a few times with a lot of text! This is normal, just ignore it!)
